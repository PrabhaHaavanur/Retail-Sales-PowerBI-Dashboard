# Retail Sales Analysis Dashboard (Power BI)

## Project Overview

This project presents a Retail Sales Analysis Dashboard created using Power BI. The dashboard analyzes sales performance, store performance, product categories, and customer purchasing behavior.

## Dataset

The dataset includes:

* Transactions
* Customers
* Products
* Stores
* Date Table

## Data Model

A Star Schema data model was created:

* Transactions (Fact Table)
* Customers (Dimension)
* Products (Dimension)
* Stores (Dimension)
* Date Table (Dimension)

## Measures Created

* Total Sales
* Total Orders
* Total Quantity
* Total Profit
* Average Order Value

## Dashboard Visuals

* KPI Cards
* Sales by Category (Bar Chart)
* Sales by Store (Column Chart)
* Monthly Sales Trend (Line Chart)
* Payment Method Distribution (Pie Chart)
* Top Products Table
* Slicers (Year, Store, Category, Payment Method)

## Tools Used

* Power BI Desktop
* Power Query
* DAX
* Data Modeling (Star Schema)
